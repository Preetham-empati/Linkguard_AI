# Preetham's Personal Portfolio

Welcome to my personal portfolio website! This repository contains the source code for my portfolio, showcasing my skills, projects, education, industrial exposure, and creative work.

## 🌟 Overview
Designed with a modern and futuristic aesthetic, this portfolio features a dynamic user experience, custom cursors, particle backgrounds, and a highly responsive layout. It serves as a digital resume and a central hub for all my professional and creative endeavors.

## ✨ Features
- **Hero Section:** Engaging introduction with a typing effect, glitch animations, and quick access buttons.
- **About Me:** Details about my journey, including my background in Electrical and Electronics Engineering and my transition to Artificial Intelligence and Data Science.
- **Skills & Expertise:** Interactive progress bars displaying my proficiency in various technical and soft skills, as well as the tools I use.
- **Education Timeline:** A structured view of my academic history, from schooling to my current B.Tech degree.
- **Featured Projects:** Highlights of my recent work, including IoT, AI, and web development projects, with links to source code and live demos.
- **Hobbies & Interests:** A personal touch sharing my interests like sketching, traveling, building electronic projects, and more, complete with interactive image galleries.
- **Certifications & Volunteering:** Showcasing my professional certifications and community involvement, such as NCC.
- **Interactive Rating System:** Allows visitors to rate my portfolio and leave reviews.
- **Contact:** Links to my email, LinkedIn, and GitHub profiles for easy communication.

## 🛠️ Technologies Used
- **Frontend:** HTML5, CSS3, JavaScript (Vanilla)
- **Typography & Icons:** FontAwesome, Google Fonts (Poppins)
- **Design Elements:** Custom CSS Animations, Flexbox, Grid, Glassmorphism, Responsive Design

## 🚀 Getting Started

To view the project locally:

1. Clone this repository:
   ```bash
   git clone https://github.com/Preetham-empati/preetham-s_portfolio.git
   ```
2. Navigate to the project directory:
   ```bash
   cd preetham-s_portfolio
   ```
3. Open `index.html` in your preferred web browser. No local server is required to view the static site.
